Jack Compiler & Language Syntax

3.1 Machine Language (Layer 0)

09

The "Ground Floor" of software.

Binary Instructions: 16-bit strings like

0000000000000010 (A-instruction) or

1110110000010000 (C-instruction).

Characteristics: Directly executed by

hardware; extremely tedious for

humans; zero abstraction.

The "DNA" of the computer

3.2 Assembly Language (Layer 1)

10

Symbolic Representation: @sum instead of @16.

The Assembler’s Job:

Like using a contact name (e.g., "Mom") in your

phone instead of memorizing her 10-digit phone

number.

  • Translate symbols to addresses.
  • Convert mnemonics (D=M) to binary.
  • Resolve labels and output .hack files.

3.3 Virtual Machine (Layer 4)

11

Intermediate Abstraction: A stack-based model.

VM Commands: push constant 5,

add, pop local 0.

Why VM?

Platform Independence: Write

once, run on any hardware with a

VM translator.

Simplifies Compiler: The Jack

compiler only needs to know how to

talk to the VM.

Primitive Types & Scope

4.1 Jack Language (Layer 6)

12

High-level, Object-Oriented.

Features: Classes, methods, fields, and

static variables.

Role: The bridge between human logic and

the VM translator.

var int result;

return result;

class Main {
function int addPair(int x, int y) {
let result = x + y;
}
}

4.2 The Operating System (Layer 6)

15

Jack OS Library: 8 Core Modules.

Math.jack (Mult/Div), Output.jack (Text),

Screen.jack (Graphics), Memory.jack

(Heap), String.jack, Keyboard.jack,

Sys.jack

The "Library" Model: In Nand2Tetris, the OS is a

set of functions called directly by your

program—no complex kernel/user mode

separation.

4.4 Jack vs Java/C

17

Similarities: Statically typed, OO

structure, curly braces.

Differences: No inheritance, no

interfaces, very simple memory

management, no garbage collection

(in the basic version).

Let & Do Statements

5.1 Jack File Structure

18

Rule: One class per file. Filename

must match ClassName.

Structure: Class declaration ->

Fields/Statics -> Constructor ->

Subroutines.

5.2 Primitive Data Types

19

int: 16-bit signed (-32768 to 32767).

char: ASCII character.

boolean: true or false.

Arrays & Strings: Handled as

objects using Array.new(size)

and String.new(length).

5.3 Variable Types & Scope

20

Field: One per object instance

(State).

Static: Shared by all instances of a

class.

Local: Exists only during

subroutine execution (Stack).

26

expressions.

6.3 Control Flow: While

27

The only loop construct in Jack!

// Display the result

do Output.printString("Sum of array elements: ");

do Output.printInt(sum);

do Output.println();

Syntax: while (condition) {
... }.
let sum = 0;
let i = 0;
while (i < length) {
let sum = sum + arr[i];
let i = i + 1;
}
